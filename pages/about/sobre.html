Sobre

Esse site foi criando por Anderson Sant'Anna, tenho 19 anos, moro em Cotia-sp, estutante de Ciência da computação na Unip de Alphaville, atualmente estou no 4º semestre, desenvolvo sistemas desktops, mobile e web, alias esse site foi eu mesmo que desenvolvi, utilizo a linguagem Java por ser uma grande linguagem de programação até hoje no mercado de trabalho, que particulamente gosto de desenvolver utilizando-a, mas estou aprendendo ainda a desenvolver site com o Java, por enquanto uso HTML 5 e CSS 3, e frameworks do CSS como o famoso Bootstrap. Eu faço cursos online pela Udemy, onde eu ganhei muitos conhecimentos, principalmente para desenvolver em android e criar Queries de banco de dados. atualmente estou fazendo curso de web lá, quero aprender usar JavaScript para o front-end e usar o Java para o back-end, assim criar um sistema com os meus conhecimentos. Gosto de aprender coisas novas estou sempre pesquisando sobre as novidades do mercado, seja por foruns ou por podcasts. Sou um aluno esforçado e como desenvolvedor também, por isso pode me contrar que não decepcionar pois eu amo o que faço! Tenho sonho de abrir minha propria empresa de software, mas primeiro quero ganhar expêriencia para alcançar esse objetivo.

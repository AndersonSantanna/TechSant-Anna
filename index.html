Por que eu escolhi Java ?

Java, uma das linguagem mais usada no Brasil, usamos ela no dia a dia sem nem notarmos, uma linguagem que roda em qualquer computador e que sua "filosofia de vida" é escreva uma vez e rode qualquer lugar. Ao mesmo tempo ela tenta economizar o maximo de codígo possível, o que um programador iniciante não se preocupa tanto com isso mas com a expêriencia que ele vai ganhando com seus projetos ele acaba se acostumando com essa reduçao de código, procurando sempre otimizar o que ja esta otimizado. Atualmente já ultizei algumas de suas APIs, como JavaFx, JPA, JSP. Estudo mais para aprender JPA e JSP para desenvolver em web com o Java, pois quero aprender outros Frameworks como o spring. Tenho o objetivo de desenvolver algum projeto web como um ERP utilizando Java, NOSQL, eu acho melhor para quem vai desenvolver sozinho um ótimo banco de dados pois te economiza tempo desenvolvendo a estrutura do que o SQL convencional.
